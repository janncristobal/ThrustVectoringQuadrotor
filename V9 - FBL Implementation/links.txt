Check git branch:
git branch

Set origin:
git remote add origin https://github.com/AVL-TMU/learn_git.git
fatal: remote origin already exists.

To create a branch and switch to it:
git checkout -b branchname

To add all files to be tracked:
git add .

To commit with a message:
git commit -m "insert your message"

To add a branch:
git push origin branchname

To switch to an exisiting branch:
git checkout branchname

To revert a branch:
git revert branchname

To stash changes: - keep the change for later, or temporarily hide them. 
git stash

To unstash:
git stash pop -> the latest stash gets pushed. 
git stash save

To delete a branch:
git push origin --delete branchname
git branch -D localbranchname

To get the latest changes:
git pull origin branchname

To find current remote URL:
git config --get remote.origin.url
